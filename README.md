Este trabajo modela la probabilidad de extinción de incendios por ondas sonoras usando mo-
delos aditivos generalizados (MAG) frente a un modelo lineal generalizado (MLG). Se empleó el
Acoustic Extinguisher Fire Dataset, con más de 15,000 observaciones de un diseño controlado con
respuesta binaria. El diagnóstico reveló relaciones no lineales y problemas críticos de concurvi-
dad en el modelo saturado. Mediante un proceso iterativo por máxima verosimilitud restringida
(REML), se removieron por redundancia las variables de decibelios y flujo de aire. El MAG final
superó estadísticamente al MLG en bondad de ajuste. Aunque el machine learning exhibe precisión
marginal superior, el modelo propuesto alcanza un accuracy del 94.0 % y preserva la interpretabi-
lidad semiparamétrica para la interpretación del usuario
